Google Checkout Java SDK - Release Notes

Changes from 0.8.9 to 2.5.0

  1. Rewrote most of the SDK, making it easier to use and maintain.
  2. Domain objects are now generated from Checkout's XSD, so they will be more complete and will stay up to date with new features.
  3. Use builder pattern to build and post commands to Checkout.
  4. Simple, extensible framework for receiving notifications.
  5. This change is not backwards compatible, so we will keep the 0.8.9 branch around.
